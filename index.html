wolicious by tholen

Wake on Lan Monitor!

Wolicious is an wake on lan application, which shows if hosts are alive or down. You can wake up down machines with one click. It is specially useful for remote lan access.

Dependencies

  • Perl 5.8.1 or greater,
  • Mojolicious,
  • Net::Ping,
  • Net::Wake,

Install

install perl 5.8.1 or greater 

$ cpan mojolicious
$ cpan Net::Ping
$ cpan Net::Wake
$ wget http://github.com/tholen/wolicious/tarball/master
$ tar xvzf tholen-wolicious-9aba6a2.tar.gz
$ cd tholen-wolicious-9aba6a2/
$ perl wolicious.pl daemon
